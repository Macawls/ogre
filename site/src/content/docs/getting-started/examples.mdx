## Rendering

```bash title="CLI"
ogre --render card.html --output card.png --format png
```

```bash title="curl"
curl -X POST https://your-server/render \
  -H "Content-Type: application/json" \
  -d '{"html":"<div class=\"flex w-full h-full bg-slate-900 p-16\">...</div>","format":"png","width":1200,"height":630}' \
  --output card.png
```

```go title="Go Library"
result, _ := ogre.Render(html, ogre.Options{
    Width:  1200,
    Height: 630,
    Format: ogre.FormatPNG,
})
os.WriteFile("card.png", result.Data, 0644)
```
